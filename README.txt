  Student Management System (CRUD)

Description

This is a console-based application built using C# and SQL Server that performs basic CRUD operations on student records.

Features Include

- Add Student
- View Student
- Update Student
- Delete Student
- Input validation and error handling

Technologies used

- C#
- SQL Server
- ADO.NET

How to run

- Open the project in Visual Studio
- Update the connection string with your SQL Server name
- Run the application
